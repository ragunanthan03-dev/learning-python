# 🧮 Simple Calculator

A simple Python command-line calculator that performs basic arithmetic operations. It allows users to enter two numbers, choose an operation, and displays the result.

---

## ✨ Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- ⚠️ Prevents division by zero
- ✅ Handles invalid operation input

---

## 🛠️ Technologies Used

- 🐍 Python 3

---

## 🚀 How to Run

1. Clone or download this repository.
2. Open a terminal in the project folder.
3. Run the program:

```bash
python calculator.py
```

---

## 🎮 How to Use

1. Enter the first number.
2. Enter the second number.
3. Choose an operation:
   - `+` Addition
   - `-` Subtraction
   - `*` Multiplication
   - `/` Division
4. The program displays the result.

---

## 💻 Sample Output

```text
Enter the first number: 20
Enter the second number: 5
Enter the operation you want to perform (+, -, *, /): /

The result is: 4.0
```

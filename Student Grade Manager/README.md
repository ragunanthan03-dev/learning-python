# 🎓 Student Grade Manager

A simple **Python** project that manages student grades using a **dictionary**. This project was built to practice Python fundamentals such as functions, loops, dictionaries, `match-case`, and exception handling.

---

## ✨ Features

* ➕ Add a student and their grade
* 📋 View all students
* 📊 Calculate the average grade
* 🏆 Display the highest grade
* ✅ Validate grades (0–100)
* ⚠️ Handle invalid input using `try`/`except`
* 🚪 Exit the program safely

---

## 🛠️ Technologies Used

* 🐍 Python 3
* 📚 Dictionaries
* 🔄 While Loops
* 🧩 Functions
* 🔀 Match-Case
* 🚨 Exception Handling (`try`/`except`)
* ✨ F-Strings

---

## ▶️ How to Run

1. 📥 Clone or download this repository.
2. 🖥️ Open a terminal in the project folder.
3. ▶️ Run the program:

```bash
python main.py
```

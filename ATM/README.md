# 🏧 ATM System using Python

A simple **ATM System** built using **Python** that demonstrates the use of **match-case (switch statement)**, functions, loops, and basic user input handling.

---

## ✨ Features

- 💰 Check Account Balance
- ➕ Deposit Money
- ➖ Withdraw Money
- 🚫 Prevents invalid deposits
- 💳 Prevents withdrawing more than the available balance
- 🔄 Menu-driven program using `match-case`
- 🚪 Exit option

---

## 🛠️ Technologies Used

- 🐍 Python 3.10+
- 🖥️ Match-Case Statement
- 🔁 While Loop
- 📦 Functions
- 🌍 Global Variables

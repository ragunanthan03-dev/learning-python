# Python MCQ Game 🐍

A simple, interactive command-line Multiple Choice Question (MCQ) game built in Python. This project tests general knowledge, records user input, and calculates a final percentage score.

## ✨ Features

* **Interactive Gameplay:** Read questions and type your answers directly in the terminal.
* **Instant Feedback:** Tells you immediately if your guess was correct or incorrect.
* **Result Summary:** Displays a side-by-side comparison of your guesses versus the correct answers at the end of the game.
* **Score Calculation:** Automatically calculates and displays your final score as a percentage.

# ✊✋✌️ Rock Paper Scissors Game

A fun and interactive **Python command-line game** where you play **Rock, Paper, Scissors** against the computer. Test your luck and strategy to see if you can beat the computer!

---

## 🎯 Features

- 🤖 Computer randomly selects Rock, Paper, or Scissors.
- 🎮 User-friendly command-line interface.
- ✅ Input validation to prevent invalid choices.
- 🏆 Determines the winner based on game rules.
- 🔄 Option to play multiple rounds.
- 👋 Displays a thank-you message when the game ends.

---

## 🛠️ Technologies Used

- 🐍 Python 3
- 🎲 `random` module

---

## 🚀 How to Run

1. Clone or download this repository.
2. Open a terminal in the project folder.
3. Run the following command:

```bash
python rock_paper_scissors.py
```

---

## 🎮 How to Play

1. Enter one of the following choices:
   - ✊ Rock
   - ✋ Paper
   - ✌️ Scissors
2. The computer randomly makes its choice.
3. The winner is decided using the classic rules:
   - ✊ Rock beats ✌️ Scissors
   - ✌️ Scissors beats ✋ Paper
   - ✋ Paper beats ✊ Rock
4. Choose whether to play another round.

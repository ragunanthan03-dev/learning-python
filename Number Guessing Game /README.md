# 🎯 Number Guessing Game

A fun and interactive **Python command-line game** where the computer randomly selects a number between **1 and 100**, and your goal is to guess it with the help of hints.

---

## 📌 Features

* 🎲 Generates a random number between **1 and 100**
* 🔢 Validates user input
* 📈 Counts the total number of guesses
* ⬆️ Displays **"Too High"** if your guess is greater than the answer
* ⬇️ Displays **"Too Low"** if your guess is smaller than the answer
* 🎉 Congratulates you when you guess the correct number
* 🚫 Handles invalid inputs gracefully

---

## 🛠️ Technologies Used

* 🐍 Python 3
* 🎲 `random` module

---

## 🚀 How to Run

1. Clone or download this repository.
2. Open a terminal in the project folder.
3. Run the following command:

```bash
python guess_number.py
```

---

## 🎮 How to Play

1. The computer randomly selects a number between **1** and **100**.
2. Enter your guess when prompted.
3. The game will provide hints:

   * 📉 **Too Low** – Guess a higher number.
   * 📈 **Too High** – Guess a lower number.
4. Keep guessing until you find the correct number.
5. 🏆 The game displays the total number of guesses once you win.

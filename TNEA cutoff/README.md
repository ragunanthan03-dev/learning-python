# 🎓 TNEA Cutoff Calculator

A simple Python program that calculates the **TNEA Engineering Cutoff** based on the user's marks in **Mathematics, Physics, and Chemistry**. It also predicts the counselling round, checks eligibility for special reservation, and determines eligibility for the **AICTE Tuition Fee Waiver (TFW)** scheme.

---

## ✨ Features

- 📊 Calculates the TNEA cutoff
- 🏫 Predicts the counselling round
- 🎖️ Checks eligibility for Special Counselling (Ex-servicemen, Sports, Disabled)
- 💰 Checks eligibility for the AICTE Tuition Fee Waiver (TFW) scheme
- 🖥️ Simple command-line interface

---

## 🛠️ Technologies Used

- 🐍 Python 3

---

## 🚀 How to Run

1. Clone or download this repository.
2. Open a terminal in the project folder.
3. Run the program:

```bash
python tnea_cutoff_calculator.py
```

---

## 🎮 How to Use

1. Enter your marks for:
   - Mathematics
   - Physics
   - Chemistry
2. Enter whether you are eligible for a special reservation.
3. Enter your annual family income.
4. The program will display:
   - Your TNEA cutoff
   - Your counselling round
   - Special counselling eligibility
   - TFW eligibility

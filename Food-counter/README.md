# 🍔 Food Counter Program

A simple Python command-line program that simulates a food ordering system. Users can select items from a menu, add them to their cart, and view the total bill at the end.

---

## ✨ Features

- 📋 Displays a food menu with prices
- 🛒 Allows users to add multiple food items
- 💰 Calculates the total bill automatically
- 🚪 Exit option to finish the order
- 🧾 Displays the ordered items and total amount

---

## 🛠️ Technologies Used

- 🐍 Python 3

---

## 🚀 How to Run

1. Clone or download this repository.
2. Open a terminal in the project folder.
3. Run the program:

```bash
python food_counter.py
```

---

## 🎮 How to Use

1. View the menu displayed on the screen.
2. Enter the name of a food item to add it to your order.
3. Repeat until you've finished ordering.
4. Type **Exit** to complete your order.
5. The program displays your order and the total amount to pay.
